# Cash Flow & Budget Analysis Tool

A simple analysis tool that helps users understand how their money is spent and suggests better ways to manage it.  
The tool works for both **personal budgets** and **company budgets** and provides clear insights using charts and summaries.

---

## What This Project Does

This project analyzes income and expenses to:
- Break spending into clear categories
- Compare actual spending with healthy reference ranges
- Highlight potential risk areas
- Suggest practical improvements
- Visualize spending patterns using charts

It is designed to be **easy to understand**, even for users without a finance background.

---

## Key Features

### Personal Budget Analysis
- Classifies expenses into:
  - Necessary fixed costs
  - Necessary variable costs
  - Discretionary spending
  - Savings
- Shows how much of your income goes into each category
- Flags high fixed costs or low savings
- Suggests small, realistic improvements

### Corporate Budget Analysis
- Analyzes company spending across:
  - Fixed operating costs
  - Variable operating costs
  - Growth investments
  - Cash reserves
- Helps identify cost imbalances
- Provides insights on flexibility and stability

---

## Visualizations Included

- Expense composition (pie chart)
- Actual spending vs recommended range (bar chart with ranges)
- Cost structure breakdown
- Savings and reserve distribution
- Side-by-side comparisons for easy interpretation

---

## How It Works (High Level)

1. User selects **Personal** or **Corporate** analysis
2. Inputs income/revenue and expense details
3. Tool calculates spending ratios
4. Results are compared with healthy reference ranges
5. Charts, insights, and suggestions are generated automatically
